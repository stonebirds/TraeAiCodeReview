# 青春活力页面配色方案（明亮鲜艳）

## 目标与原则
- 明亮、鲜艳、年轻化；避免灰暗深色
- 文本与背景对比度满足 WCAG 2.1 AA（≥ 4.5:1）
- 保持统一视觉层次与品牌风格；主要颜色不超过 5 种

## 推荐色值
- 主色（二选一）：
  - 柠檬黄 `#FFD700`（L≈85，S≈100）
  - 亮青色 `#00CED1`（L≈72，S≈83）
- 辅助色（任选2~3）：
  - 珊瑚粉 `#FF6B6B`
  - 薄荷绿 `#7BED9F`
  - 天蓝色 `#87CEFA`
- 强调色：
  - 橙红色 `#FF4500`

## 使用场景
- 背景：浅色或白色 `#FFFFFF` / `#F5F5F5`
- 主文本：深色 `#333333`
- 标题/强调：更深 `#111827` 或主/强调色搭配
- 按钮与交互：主色高饱和度版本，白字或深色字取决于对比度
- 图标与装饰：使用辅助色系点缀

## CSS变量
- 文件：`src/styles/color-vars.css`
- 变量：
  - `--ui-primary: #FFD700`
  - `--ui-secondary: #00CED1`
  - `--ui-accent: #FF4500`
  - `--ui-coral: #FF6B6B`、`--ui-mint: #7BED9F`、`--ui-sky: #87CEFA`
  - `--ui-bg: #F5F5F5`、`--ui-surface: #FFFFFF`、`--ui-border: #e5e7eb`

## 应用指南
- 顶栏：主色背景 + 深色文本（`#333`）或白字，保证对比度
- 面板：白色卡片（`glass-card`），边框 `#e5e7eb`，阴影轻柔
- 输入/按钮：使用主色与强调色；聚焦环采用辅助色，保持活力
- 日志面板：浅背景下更深文本，列表项使用淡色背景区分

## 对比度校验
- 参考 `docs/contrast-report.md`；普通文本 ≥ 4.5:1，标题 ≥ 3:1
