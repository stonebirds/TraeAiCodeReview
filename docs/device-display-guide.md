# 不同设备显示效果与截图采集指南

## 设备与尺寸
- 移动：iPhone 14/15、安卓旗舰（360×800、414×896）
- 平板：iPad 11"、安卓平板（768×1024、1024×1366）
- 桌面：1080p、1440p、4K（1280×800、1920×1080、2560×1440、3840×2160）

## 截图场景
- 输入配置页：顶栏、表单、拖拽上传区、示例按钮
- 操作日志页：进度条、日志列表（含过滤与自动滚动）
- 审查结果页：统计卡片、问题列表、代码上下文

## 截图指引
- 统一深色主题背景；窗口宽度最大化
- 保证文字与背景对比清晰；日志区域高度展示完整列表
- 命名规范：`device-page-resolution.png`（示例：`iphone15-logs-1290x2796.png`）

## 采集路径建议
- 建议将截图保存至 `docs/screenshots/` 目录，按设备与页面分子文件夹归档
